git config

git config is a command in Git that is used to configure various settings for your Git repository or your user profile. It allows you to set or modify configuration options that control how Git behaves in different contexts. In a concise explanation: git config is used to set or modify Git configuration settings. example : git config --global user.name "Your Name"

git add commit - m "your updates"

git add . is a command used in the Git version control system. It's used to stage (prepare) all the changes you've made in your current working directory and its subdirectories for the next commit. Staging means that you're telling Git to include these changes in the upcoming commit snapshot. In a more concise explanation: git add . stages all changes in the current directory and its subdirectories for the next commit.
